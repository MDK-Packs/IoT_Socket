IoT Socket Documentation

Overview

The IoT Socket pack provides a simple IP Socket API (BSD like) that implements the glue logic between IoT cloud connectors and the underlying network stack.

Structure of an IoT application

Currently, an implementation for the MDK-Middleware network stack or a CMSIS-Driver WiFi is available. This makes it easy to retarget an IoT application running on an Arm Cortex-M based microcontroller device to either wired or wireless IP connections.
